Movie Selector-javafx

It is a small project for CS 2340 in Georgia Tech. The basic project for this course is a web application using J2EE.
This application will require user to log into the system and share private recommendations and comments about the movie they have seen. They are also able to search movies by title, find the recently released movies and new released DVDS.
All movie data are from Rotta Tomato's API and our own database.
This is alternative platform of the project using javaFX.
This javafx application includes all functionalities of the web application.

Movie Selector-Android

This is another platform of web application using android. This version also performs all functionalities.

PipeLined Processor

This is the project from CS2200 in Georgia Tech. In this project, we were making a pipelined processor with five different stages: FETCH, INSTRUCTION DECODE/REGISTER READ, EXECUTE, MEMORY, and WRITE BACK.
